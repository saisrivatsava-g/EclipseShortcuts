# EclipseShortcuts
eclipse shortcuts

### Opening a class
```CTRL+SHIFT+T```

### To search for a file
```CTRL+SHIFT+R```

### Quick Outline
```CTRL+O```

### Open Type Hierarchy
```F4```

### Full text search
```CTRL+H```

### Jumping to Class declaration
select the class name and press ```F3``` or ```CTRL+ mouse left click```

### To open terminal in eclipse
```CTRL+ ALT+ T```

### Quick fix
```CTRL+ 1```
